# Data Visualization Dashboard

## Overview
This project is a data visualization dashboard built using React and Recharts. It demonstrates how different types of charts can be used to represent and interpret data in a clear and meaningful way.

## Features
- Bar chart for comparing values across categories
- Line chart for showing trends over time
- Pie chart for displaying proportions and distribution
- Interactive and responsive visual components
- Clean and structured UI layout

## Purpose
The goal of this project was to practice:
- Data visualization using React
- Integrating chart libraries (Recharts)
- Representing data in different visual formats
- Improving UI design for dashboards

## Technologies Used
- React
- Recharts
- JavaScript
- CSS

## How to Run
1. Clone the repository  
2. Run `npm install`  
3. Run `npm start`  
4. Open in browser at `http://localhost:3000`

## What I Learned
- How to structure and pass data into chart components
- When to use different chart types depending on data
- How to build reusable and responsive UI components
